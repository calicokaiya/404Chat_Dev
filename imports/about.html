What is 404Chat?

404Chat is a web application private chat, where every single one of your messages is deleted a couple of seconds after being sent, and everything is deleted from the database twice a day.
The point is to offer a safe, private and temporary environment to communicate important and possibly classified information to others.

Who made 404Chat?

My contacts will remain hidden for now, but I do plan on coming out and showing myself to some extent some day. I'm just not ready for it yet.

How do I use 404Chat?

This application works through the use of mainly chat commands. These commands are:

  • "/create [roomname] [password]". This will create a room with name [roomname] and password [password].
    Password is optional.
    If you try to create a room that already exists, it will attempt to join said room with provided roomname and password.
  • "/join [roomname] [password]". This will join a room if the roomname and password combinations are valid.
  • "/help" displays this information in the chatroom so you don't have to visit the about page constantly.
